Summary

Handles the installation of all required CUPS packages

Overview

Private class

This class is a convenience wrapper around a package resource. It inherits the package_* attributes from the public puppet_classes::cups class and by default ensures their presence.

Examples:

This class is implicitly declared through

class { '::cups':
  package_names => ['cups', 'ipptool']
}
